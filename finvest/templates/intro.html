A comprehensive web-based personal finance management application built with Flask.

Get Started Learn More

Project Overview

FinVest features transaction tracking, portfolio management, budgeting, and detailed financial reports with interactive charts. The project integrates MySQL for database management, applies modern web development practices, and follows clean architecture principles.

"The system manages financial transactions, investment portfolios, budgets, special events, and generates insightful financial analytics with persistent database storage."

Project Objectives

Modern Tech

Manage personal finances using modern web technologies

Data Viz

Apply data visualization for financial insights

MySQL Storage

Store and retrieve financial data using MySQL

Glassmorphism

Provide intuitive user experience with glassmorphism design

Project Features

Transaction Management

Record and categorize all financial transactions with advanced filtering and real-time updates.

Portfolio Management

Monitor stocks, cryptocurrencies, and mutual funds with performance analytics and price history.

Budgeting

Set monthly and category-specific budgets with alerts and spending analysis.

Event-Based

Create isolated budgets for special events like vacations or weddings with date range management.

Reports & Analytics

Beautiful interactive charts generated with Matplotlib with glassmorphism theme.

Modern UI/UX

Translucent UI with backdrop blur, dark/light mode, and smooth animations.
